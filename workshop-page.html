DE WORKSHOP

1. UITLEG PROGRAMMA P5.JS

In deze tutorial laten we zien hoe het Programma werkt waar de kinderen in Gaan coderen. Denk hierbij aan het Inloggen, openen van bestanden, de Werking van de Setup en draw. Het Runnen en stoppen in p5.js

2. BASISVORMEN MAKEN

Nadat de uitleg over het programma P5.JS is besproken kunnen we met het programma aan de slag. Hierin leren we de benamingen van de basisvormen in code en deze op correcte wijze te plaatsen.

3. KLEUREN

In het vorige onderdeel heb je geleerd hoe je gebruik maakt van vormen in p5.js. In deze tutorial leer je hoe je gebruik maakt van kleuren zodat de vormen die je hebt gemaakt veel unieker zijn. Je leert hoe je gebruik maakt van RGB waardes in p5.js. Zo kun je jou code veel specifieker maken.

4. SPEURTOCHT

De cursus wordt afgerond met een Interactief onderdeel. Hierin gaan de kinderenEen speurtocht doen, hoe deze in zijn werkingGaat wordt uitgelegd in deze tutorial. Zo kraken zij de code om de slechterik te verslaan.
